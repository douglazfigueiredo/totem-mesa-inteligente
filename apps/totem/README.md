# @app/totem

PWA do totem (tablet landscape, 1 por mesa). Esqueleto reservado — scaffolding ocorre na **Fase 3**.

Stack-alvo:

- Next.js 16 (PWA, modo kiosk)
- Zustand (estado client-side)
- Dexie / IndexedDB (cart offline)
- Socket.IO client (conexao com hub local)
- Tokens: `@app/ui/tokens`
- Schemas: `@app/schemas`
